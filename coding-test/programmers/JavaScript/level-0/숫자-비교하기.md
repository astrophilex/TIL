## JavaScript lv.0

정수 `num1`과 `num2`가 매개변수로 주어집니다. 두 수가 같으면 1 다르면 -1을 retrun하도록 solution 함수를 완성해주세요.

```javascript
function solution(num1, num2) {
    if(num1==num2){
        var answer = 1;
        return answer;
    }else{
        var answer = -1;
        return answer;
    }
}
```
